# Отладка


## Уровни навыка
### Уровень 1
Умеет пользоваться базовыми инструментами для отладки web-приложения через браузер (на примере Chrome):

* Elements (отладка верстки и стилей)
* Network (просмотр запросов)
* Console (для отладки кода с помощью console.log)
* Application (работа с куками и кэшем приложения)
* Device Bar (для тестирования приложений на разных разрешения)

### Уровень 2
Может отладить код с помощью браузера:

* умеет ставить брейкпоинты в коде с помощью debugger
* умеет ставить брейкпоинты в браузере (line, conditional, xhr, dom, event listener, exception, function)
* знает, как воспользоваться блоками Watch, Call Stack, Scope
* может воспользоваться пошаговым выполнением кода

Владеет dev tools для фреймворков (в зависимости от стека):

* React Dev Tools
* Angular Dev Tools

### Уровень 3
Владеет инструментами для модифицирования запросов:

* умеет блокировать запросы
* может модифицировать ответ запросы при помощи инструментов мокирования (charles, chrome tools и др)

Владеет отладкой на реальных мобильных устройствах через инструменты браузер

Может провести базовый performance аудит и выявить проблемы с производительностью приложения (вкладка Audits, Lighthouse)

### Уровень 4
Владеет профилированием js приложений, способен проанализировать runtime приложения, выявить проблемы с cpu 

Может выполнить отладку nodejs приложения, выявить проблемы с производительностью (проанализировать CPU Usage, Heap Usage, найти причину утечки памяти)
