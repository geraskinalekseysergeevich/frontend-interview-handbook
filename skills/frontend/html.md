## Уровни навыка
### Уровень 1
* HTML
  * теги
  * блочные и строчные элементы
  * атрибуты тегов
  * meta-теги
* CSS
  * способы подключения
  * селекторы
  * псевдо-селекторы (:before, :after)
  * псевдоклассы состояний (:hover, :focus, :active, .etc)
  * свойства и значения свойств

*Умение*: Способен сверстать типовые конструкции:
* виджет товара в интернет-магазине
* пост в блоге с комментариями
* форму авторизации
* таблицу с ценами на услуги по верстке

### Уровень 2
* HTML
  * семантическая разметка
* CSS
  * работа с переменными
  * работа с медиа запросами
  * работа с кастомными шрифтами
    * подключение и использование
  * CSS-анимации
    * понимание влияния на производительность
* блочная модель
* понимание потока
* позиционирование
* контекст наложения
* доступность
* адаптивная верстка
* SVG
  * понимание как оптимизировать и убрать лишние теги

*Умение*: Способен сверстать:
* Лендинг
* Сайт-визитку с несколькими страницами
* Шаблон интернет магазина

### Уровень 3
* CSS-методологи
* CSS-фреймворки
* пре/пост-процессоры (PostCSS, SASS, Less), для упрощения написания кода
  * переменные
  * миксины
* компонентный подход (CSS Modules, Styled Components)

*Умение*: Способен организовать дизайн-систему в коде
