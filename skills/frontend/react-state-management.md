# React — State-management

## Уровни навыка
### Уровень 1
Выбирает правильное применение локальному стейту (useSate), либо данным связанным с компонентой (useRef). Не допускает ошибок по типу SSOT. Может без ошибок обработать получение и хранение данных от сервера (загрузка, гонка). Не хранит вычисляемые данные в стейте приложения без обоснованной необходимости.

### Уровень 2
Понимание концепций, стоящих за популярными библиотеками управления состоянием: redux, mobx и т.д. Понимание как реализована интеграция таких систем с реактом. Особенности работы и caveats: проблемы с производительностью редакса и пути их решения (reselect), stale props, zombie children. Следит за перспективными решениями (effector, swr, react query)

### Уровень 3
Знаком с такими концепциями как state persistence, data normalization, immutable structures, работа с кешем, data rehydration. Применял localStorage, sessionStorage для решения релевантных задач.

### Уровень 4
Имеет опыт хранения и обработки большого объема дынных на клиенте (indexDB и прочие). Имеет опыт работы с распределенным стейтом. Может построить систему с синхронизацией данных посредством протоколов (сеть, воркеры и т.д.)
