## Уровни навыка
### Уровень 1
Понимает основные системы, которые необходимы для работы текущего приложения на котором работает сотрудник и как эти системы взаимосвязаны между собой.

### Уровень 2
Может спроектировать небольшой сложности типовой проект, адаптировав незначительные моменты под нужды проекта. Может собрать функциональные требования для сервиса.

### Уровень 3
Полностью разбирается как работает текущий проект, на котором работает, какие есть слабые места у текущего решения и может спроектировать новый сервис внутри действующий системы, который будет возможно встроить в текущую архитектуру и инфраструктуру. Может собрать не функциональные требования для сервиса и знает об ограничениях системы.

### Уровень 4
Может спроектировать целые системы с множеством взаимосвязанных компонентов/сервисов/команд с возможностью встроить в нашу текущую инфраструктуру.

### Уровень 5
Может спроектировать сложную систему используя и понимания системы из соседних стеков (другие языки, платформы, базы данных, сервисы).
