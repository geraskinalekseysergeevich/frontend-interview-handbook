## Как пользоваться

1. Определи свой текущий уровень в матрице
2. Кликни на нужный навык — откроется страница с теорией
3. Читай по разделам, изучай примеры кода
4. В конце каждой страницы — источники для углублённого изучения

## Уровни знания

| Уровень | Значение |
|---------|----------|
| **0** | Не требуется |
| **1** | Базовое понимание и применение |
| **2** | Уверенное владение, понимание internals |
| **3** | Глубокая экспертиза, системный подход |
| **4** | Экспертный уровень, архитектурные решения |
